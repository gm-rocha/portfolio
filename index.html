Como trabalho

Entendo o problema antes dos números

Antes de abrir qualquer Power BI ou rodar uma query, eu paro e penso: "Qual o real desafio aqui?"
Converso com o time, entendo os objetivos, o contexto e o impacto.
Resolver por resolver não basta — é sobre acertar onde realmente importa.

Mergulho nos dados e organizo o caos

Hora de colocar a mão na massa.
Reúno os dados das fontes certas, limpo, trato, transformo. Uso Power Query, SQL ou Python, dependendo do cenário.
Meu foco aqui é deixar os dados prontos pra falar — sem ruído, sem bagunça.

Busco padrões, tendências e o "insight escondido"

Com tudo em ordem, começa a parte que eu mais curto: explorar.
Visualizo os dados, cruzo as informações, sigo perguntas que vão surgindo… até encontrar aquele insight que muda o jogo.

Portfólio

PBI | Painel Gerencial

Um dashboard executivo projetado para facilitar a leitura rápida dos principais dados de operação, reunindo em uma única página os indicadores mais estratégicos do negócio.
Com layout objetivo e filtros inteligentes, o painel destaca variações críticas em indicadores como cobertura, share, ruptura, preço e ponto extra — ajudando a gestão a focar no que realmente importa: onde estão os problemas e onde estão as oportunidades.

SQL | Cálculo de Mix de Produtos

Com dezenas de SKUs por loja e múltiplas fontes de coleta, definir o mix correto de produtos era uma tarefa demorada, suscetível a erros e difícil de escalar. Este script SQL veio para resolver isso: consolida tudo em uma única base integrada, identifica a fonte por SKU, classifica o status de disponibilidade e entrega um mix limpo, padronizado e pronto para análise.

PBI | Painel de Indicadores

Mais do que metas e resultados: este dashboard revela onde está a força (ou fraqueza) da execução comercial.
De forma visual e orientada por dados, ele permite enxergar a performance sob três ângulos essenciais — Sell In, Sell Out e Positivação — filtrando, comparando e destacando o que realmente precisa de atenção.

Python | Localizador de Dados

E se você pudesse vasculhar dezenas de arquivos em segundos e saber, com precisão, em qual deles estão os CNPJs que importam?
Foi com essa proposta que desenvolvi este script: uma ferramenta simples, mas extremamente útil para localizar rapidamente parceiros ou pontos de venda em meio a um mar de planilhas.

PBI | Análise de Vendas

Dashboard criado para transformar dados dispersos em respostas rápidas sobre o desempenho comercial.
Em uma única página, ele consolida vendas por loja, produto e período — permitindo entender o que mais vende, onde, quando e por quê.
Visual claro, permitindo uma leitura rápida para guiar decisões comerciais com mais precisão.

Python | Consolidando Dados

Em meio a dezenas de planilhas com dados de vendas espalhados, surgiu a necessidade de consolidar tudo em uma única base limpa, padronizada e pronta para análise — mesmo com as informações úteis limitadas a algumas abas.
Este script em Python automatiza todo o processo: abre, extrai, padroniza e empilha os dados, economizando tempo e garantindo consistência para análises futuras.
